# HTMLMediaElement.getStartDate() demo

This example demonstrates how to use the [`HTMLMediaElement.getStartDate()`](https://developer.mozilla.org/en-US/docs/Web/API/HTMLMediaElement/getStartDate) method to retrieve the real-world date and time corresponding to the beginning of an HLS stream. The stream's `#EXT-X-PROGRAM-DATE-TIME` tags provide the date that `getStartDate()` returns.

[See the example live](https://mdn.github.io/dom-examples/media/getstartdate/).
